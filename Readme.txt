Shortcut for display :
F1 - Help
1 - Display the current playlist
2 - Filesystem browser
3 - Search in databases
4 - Music library
5 - Playlist editor
6 - Tag editor
7 - Output selector
8 - Music visualizer
= - Wall clock
Space/Spasi - Change visualizer

Other UI Shortcuts :
q - Exit
f - Seek forward
b - Seek backward
\ - Change the classic / alternative display
# - Display the bitrate on the song
i - Display song info
l - Displays the lyrics for the song being played
L - Search for more lyrics found in the database
p - Play / Pause
u - then press ‘u’ in order to update the database in music update
Shift++ - Adds volume
- - Reduce volume
